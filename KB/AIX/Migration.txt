[ ] **Perform the migration using screen** session(in case the connection breaks down):
	* to create a session
		screen -S migration
	* to list the sessions
		screen -list
	* to dettach to a session
		screen -d
	* to attach to a session
		screen -r <session_name>
	* to attach to an already created session
		screen -x <session_name>
	* to remove a dettached session
		screen -XS <session_name> quit
