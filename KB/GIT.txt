====== GIT ======
Created Sunday 26 January 2014

**Set up computer 1**
 git init  
 git pull git@github.com:username/git-repo.git  
 git remote add github-repo git@github.com:username/git-repo.git  

**Set up computer 2**
 git init  
 git pull git@github.com:username/git-repo.git  
 git remote add github-repo git@github.com:username/git-repo.git  

Make changes on computer 1 and push the changes to the central repository
 edit files  
 git commit -m "commit changes"  
 git push github-repo  

**Now get these changes on computer 2** 
 git pull github-repo
